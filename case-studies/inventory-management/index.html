The Challenge

Their legacy system was bleeding time and money

Every event launch felt like a crisis. Staff panicked. Systems crashed. Data didn't match. Sound familiar?

CSV Imports Crashed Before Events

When brand manifests arrived with 3,000-10,000 SKUs, the system would choke. Time-critical uploads before events caused timeouts, crashes, and panic. A single invalid row killed the entire import.

No Real-Time POS Sync

Manual data entry between the warehouse system and Square POS meant hours of lag. Inventory counts never matched. Staff wasted time on duplicate entry across multiple systems.

Event Setup Took 4+ Hours

With 1M+ SKUs in the library but only 100K needed per event, there was no way to sync selectively. Every event required hours of manual adjustments and no visibility into event profitability.

No Visibility Into Errors

When imports failed, nobody knew why. Generic error messages left analysts guessing. Developer intervention was required hours before critical events just to figure out what went wrong.

The Solution

We built a system designed for high-volume operations

A domain-driven architecture that handles massive scale without breaking, with seamless Square POS integration.

Queue-Driven CSV Imports

Async processing with 1,000-row chunking. Upload 10,000 SKUs in under 3 minutes. Row-level error feedback shows exactly what failed — no more mystery crashes.

Two-Way Square Sync

30-minute automated sync intervals. Push catalog changes to Square, pull sales transactions back. 99.7% accuracy across all locations — no more manual reconciliation.

Event-Scoped Operations

Selective syncing means only relevant SKUs (100K of 1M) go to each location. Event setup dropped from 4+ hours to under 30 minutes. Track profitability by event.

Bulk Pricing Engine

Apply "65% OFF MSRP" across thousands of items instantly. MySQL stored procedures handle bulk updates in seconds. No more item-by-item price changes before events.

Real-Time KPI Dashboard

Total inventory value, sales trends, low stock alerts — all at a glance. Built with shadcn-vue for fast, responsive interfaces that work on warehouse tablets.

Audit Trail & Error Logs

Every import logged with metrics. Row-level error messages with line numbers. Finally, complete visibility into what's happening — no more developer panic calls.

The Results

From panic to confidence

The system launched successfully for the February 2025 event cycle with zero import failures across 3 warehouse locations.
